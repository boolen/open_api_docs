客户订单

查询订单的API

调用请求

GET /v1/deals?access_token={access_token}

参数说明

  • {access_token}用从“获取身份权限”节拿到的access_token替换
  • 除了access_token参数外，该API还支持以下参数来缩小查询结果集:
    • customerId 查询指定客户的订单
    • rows 每页的记录数
    • page 第几页
    • sidx 排序的字段
    • sord 排序的方式, asc是升序，desc是降序
